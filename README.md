# Сайт-поздравление для Кати Гнома

## Запуск

Нужен установленный [Node.js](https://nodejs.org) (версия 18+).

```bash
npm install      # один раз — поставить зависимости
npm run dev      # запустить локально (откроется на http://localhost:5173)
```

Для готового сайта:

```bash
npm run build    # собрать в папку dist/
npm run preview  # посмотреть собранную версию
```

## Как менять данные (для своих дней рождения)

Всё лежит в одном файле: **`public/data.json`**. Менять можно без знания кода.

```jsonc
{
  "name": "Кати",                 // имя в заголовке
  "short": "Кати",                 // короткое имя (в подписях, если нужно)
  "subtitle": "воспоминания…",    // подзаголовок (можно оставить пустым "")
  "photos": [
    {
      "src": "foto1.jpg",         // ссылка или имя файла картинки ("" = заглушка)
      "caption": "Текст на карточке",
      "author": "Кто добавил",
      "text": "Длинный текст, который появится по клику на фото. Можно оставить пустым."
    }
  ]
}
```

- **caption** — короткий текст, который виден прямо на полароиде.
- **text** — длинный текст; появляется в окошке при нажатии на карточку. Если пустой `""` — карточка просто открывает фото покрупнее.
- **src** — картинки удобнее всего положить в папку `public/` рядом с `data.json` и писать имя файла, например `"src": "foto1.jpg"`.

После `npm run dev` правки в `data.json` подхватываются автоматически
